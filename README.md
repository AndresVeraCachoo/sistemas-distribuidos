Repositorio oficial para el desarrollo de prácticas, laboratorios y proyectos de la asignatura **Sistemas Distribuidos**, impartida en la **Universidad de Burgos (UBU)**.

## 📂 Estructura del Repositorio

El contenido está organizado en carpetas independientes para cada entrega o módulo de la asignatura:

| Carpeta | Descripción | Tecnologías |
| :--- | :--- | :--- |
| 📁 `/Practica-1` | **Práctica 1:** Configuración entorno de prácticas JEE | Terminal, Linux |
| 📁 `/Practica1Chat` | **Práctica 1 (Chat):** Sistema de Chat Cliente/Servidor | Java, Sockets TCP/IP |
| 📁 `/Ejercicio-2` | **Ejercicio 2:** Desarrollo y empaquetado del primer proyecto | Java, Maven |
| 📁 `/Practica-2` | **Práctica 2:** Sistema web distribuido. Spring Boot (Frontend) + API Flask (Backend) | Java, Python, PostgreSQL, Docker |

> **Nota:** Cada carpeta contiene su propio `README.md` con instrucciones detalladas, arquitectura y una breve introducción al módulo correspondiente.
