Este directorio contiene la resolución del **Ejercicio 2** de la asignatura Sistemas Distribuidos (UBU).

Se ha optado por realizar el **Punto 2** de la guía, creando un proyecto directamente integrado con Maven en Eclipse (tipo Maven Project), ya que engloba el ciclo de vida completo requerido para la práctica.

## 🎯 Objetivos

- [x] Creación de un proyecto Maven (`es.ubu.sd:ejercicio_2:1.0.0`).
- [x] Configuración del archivo `pom.xml` forzando el uso de **Java 1.8** y codificación UTF-8.
- [x] Inclusión de código fuente básico (`Principal.java`).
- [x] Compilación del proyecto.
- [x] Generación de documentación técnica (JavaDoc).
- [x] Empaquetado de los binarios en un fichero `.jar`.

## ⚙️ Estructura del Proyecto

El proyecto sigue la estructura estándar de directorios de Maven:

```text
ejercicio_2/
├── src/
│   └── main/java/es/ubu/sd/   # Código fuente (Principal.java)
├── target/                    # Archivos generados (Binarios y Documentación)
├── pom.xml                    # Configuración central de Maven
└── README.md                  # Este archivo
```
